# Agent Notes

Короткие правила для Codex/агентов в этом проекте.

- Кириллица в комментариях и разметке нормальна. Не считать русские комментарии проблемой проекта.
- Если терминал показывает кириллицу как `Рџ...`, это, скорее всего, проблема вывода/кодировки терминала, а не файла.
- При `apply_patch` не использовать русские комментарии как основной якорь контекста. Матчить по устойчивым строкам кода рядом: именам функций, id, class, вызовам методов.
- Если нужно читать или проверять русскоязычный текст, учитывать UTF-8 и особенности PowerShell/терминальной кодировки.
- Планы реализации и ответы пользователю всегда писать на русском языке.
- Для локальной проверки игры сначала использовать уже рабочий адрес `http://127.0.0.1:4173/`.
- Не поднимать новый dev/static server, если игра уже открыта, пользователь тестирует вручную или существующий адрес работает.
- Browser/visual checks делать коротко и по делу: только когда они реально нужны после изменений.
- Не продолжать долгие проверки после успешной реализации и ручного подтверждения пользователя.
- После browser-проверок не оставлять временные `.edge-profile-*` и screenshot-файлы, если они не нужны пользователю.
